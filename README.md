###################################Игра Танчики###################################  

Игра рассчитана на двоих человек.

В игре присутствует два типа препятсвий, один из которых невозможно разрушить.

Управление:
Танк 1: 
WASD + ESCAPE

Танк 2: 
←  ↑  → ↓ + ENTER
